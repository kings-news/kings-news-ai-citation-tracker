## Overview

Kings News AI Citation Tracker tracks how visible your press releases are across AI platforms, search engines, and the modern discovery ecosystem.

Simply upload your press release, article, or content piece, and the platform analyzes its presence and citation potential across leading AI assistants, search engines, and digital discovery channels.

---

## Installation

### Node.js
```bash
npm install @kings-news/ai-citation-tracker
```

### Python
```bash
pip install kings-news-citation-tracker
```

---

## Usage

### Node.js CLI
```bash
npx ai-citation-tracker "press-release-title" 72 65 80 55 70 60
```

### Python CLI
```bash
python -m tracker "press-release-title" 72 65 80 55 70 60
```

## Citation Signal Scores

| Signal | Description | Score Range |
|--------|-------------|-------------|
| AI Citation | Content suitability for AI platform references | 0–100 |
| Search Visibility | Traditional and AI-enhanced search optimization | 0–100 |
| Citation Readiness | Signals, structure, authority, context | 0–100 |
| Visibility Gap | Weaknesses preventing AI discovery | 0–100 |
| Platform Reach | Coverage across AI platforms and search ecosystems | 0–100 |
| Content Optimization | Headlines, clarity, attribution, structure | 0–100 |

## Score Interpretation

| Score | Status | Action |
|-------|--------|--------|
| 0–30 | Critical | Immediate optimization required |
| 31–60 | At Risk | Active improvement needed |
| 61–80 | Healthy | Maintain and refine |
| 81–100 | Excellent | Sustain and scale |

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| content | string | Content title or identifier |
| ai_citation | integer | AI Citation score (0–100) |
| search_visibility | integer | Search Visibility score (0–100) |
| citation_readiness | integer | Citation Readiness score (0–100) |
| visibility_gap | integer | Visibility Gap score (0–100) |
| platform_reach | integer | Platform Reach score (0–100) |
| content_optimization | integer | Content Optimization score (0–100) |
